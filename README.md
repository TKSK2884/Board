# 게시판 페이지

이 프로젝트는 사용자가 게시글을 작성하고 조회할 수 있는 서비스입니다. 
공지사항 게시판과 자유게시판으로 구분되어 있으며, 로그인한 사용자는 글을 작성할 수 있습니다. 사용자 편의성을 고려하여 게시판별로 게시글을 쉽게 조회하고 관리할 수 있도록 설계되었습니다.
